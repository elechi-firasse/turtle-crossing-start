## **Turtle Crossing Game**

A small Turtle-graphics arcade game where you guide a turtle across the road while dodging fast-moving cars. The game speeds up as you level up, making each round a bit harder.

**If you're on Windows, you can simply download the `.exe` file and run the game.**

---

## **How to Play**

* Use the **Up Arrow** to move the turtle forward
* Reach the top to level up
* Avoid all cars or the game ends

---

## **Run From Source**

```
python main.py
```
